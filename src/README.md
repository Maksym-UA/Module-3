ESP32 S3 Debounce handling in ESP-IDF framework using interrupt and timer

SETUP
=====

 1. Install PlatformIO.

 2. Install ESP-IDF extension.

 3. Open the project in VS Code

 4. To build the project, execute the command in the VS terminal
 ```
 $ pio run
 ```
 5. To upload the project to the ESP32 run this command
 ```
 $ pio run -t upload
 ```
 6. Monitor serial output at 115200 baud
 ```
 $ pio device monitor -b 115200
 ```
